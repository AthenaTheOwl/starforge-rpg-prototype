# Portfolio-grade boundary

This is an Act 1 Godot source copy from an active passion project, cleaned for
review and future iteration.

It is not a promise that the game is complete. It is evidence of game-system
architecture at real project scale: data files, scenes, scripts, UI, tests, and
validation.

The repo keeps source and test artifacts for the public slice. It excludes
local editor state, agent workflow state, and unreleased later-act content.
